# RunFlow PWA — Instrucciones de instalación

## Archivos incluidos
- `index.html` — La app completa
- `manifest.json` — Configuración de PWA
- `sw.js` — Service Worker (modo offline)
- `README.md` — Este archivo

## Publicar gratis con GitHub Pages

1. Crea una cuenta en github.com (si no tienes)
2. Crea un repositorio nuevo llamado `runflow`
3. Sube los 3 archivos: index.html, manifest.json, sw.js
4. Ve a Settings → Pages → Source: "main branch"
5. En 2 minutos tendrás una URL tipo: `https://tuusuario.github.io/runflow`

## Instalar en iPhone

1. Abre la URL en Safari (importante: debe ser Safari)
2. Toca el botón de compartir (□↑)
3. Selecciona "Agregar a pantalla de inicio"
4. ¡Listo! La app aparece como nativa en tu iPhone

## Publicar con Netlify (alternativa aún más fácil)

1. Ve a netlify.com → "Deploy manually"
2. Arrastra la carpeta runflow-pwa
3. ¡En 30 segundos tienes tu URL!
